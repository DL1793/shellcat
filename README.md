# A tamagotchi style shell companion written using the curses library.

Made for my first personal project over at [boot.dev](https://boot.dev).

The program uses a simple loop to handle input, states and rendering. Each screen is composed of a background, foreground (the shellcat character) and UI. 

You can feed the cat, play with the cat, and change the system color inside the Settings menu.

## Planned features:
- Add a store for food and toys
- Add a progress system to store shellcat stats and user currency
- Add minigames for different toys
- Add GitHub integration to allow users to convert commits to currency
